ROOM 01

The Decluttering Room

"Keep everything, and the door stays shut."

What's Inside

The walls are covered in to-do sticky notes for seminars, networking, internship prep, and workouts — and a numeric lock code is hidden somewhere among them. If you try to keep every single task, the code will never line up.

THE TWIST

Only by boldly tearing some sticky notes off the wall do the remaining tasks line up into the lock code.
